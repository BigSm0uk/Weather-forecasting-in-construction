# StroyPredict AI

Автоматизация прогнозирования рисков производства строительных работ методами машинного обучения на основе открытых метеорологических данных.

Система прогнозирует почасовую погоду (температура, ветер, осадки) с помощью моделей XGBoost, обученных на исторических данных Open-Meteo с 1940 года, и оценивает допустимость выполнения строительных работ по нормативным ограничениям.

## Дашборды

| Страница | Описание |
|---|---|
| **Дашборд данных** | Почасовой прогноз температуры, ветра и осадков на выбранную дату |
| **Тепловая карта рисков** | Оценка безопасности работ по часам с цветовой индикацией |
| **Анализ графика работ** | Загрузка CSV-расписания и анализ погодных рисков по дням с почасовой детализацией |
| **Справочник работ** | Управление видами работ и их допустимыми погодными границами |
| **Управление моделями** | Запуск обучения ML-моделей по регионам, просмотр метрик |

## Стек

- **Frontend:** Next.js, Tailwind CSS, shadcn/ui, Zustand
- **Backend (ML API):** FastAPI, XGBoost, SQLAlchemy, PostgreSQL
- **Данные:** Open-Meteo Archive API
- **Инфраструктура:** Docker Compose

## Запуск

Требуется: Docker, [Task](https://taskfile.dev), [Bun](https://bun.sh)

```bash
# Поднять бэкенд (PostgreSQL + FastAPI)
task d:build    # первый запуск (сборка образа)
task d:up       # последующие запуски

# Запустить фронтенд
task f:dev
```

Бэкенд будет доступен на `http://localhost:8000`, фронтенд — на `http://localhost:3000`.

## Формат CSV для анализа графика

```
Название работы, ID типа работы, Дата начала, Дата окончания
Бетонирование фундамента, 1, 2026-06-01, 2026-06-15
```
